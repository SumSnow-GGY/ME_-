**调速阀**
它是由单向阀和节流阀并联组成。根据单向阀开口的方向不同，可分为排气节流和进气节流两种。
**△ 排气节流：调节气缸速度容易，气缸活塞运行稳定，是常用的回路元件；**
**△ 进气节流：虽然也能控制流量变化，但难以控制速度稳定性，一般只用于单作用气缸、夹紧缸和低摩擦气缸等的速度控制。**

# 调速阀的种类


## 一、按结构原理分类（通用）
| 类型 | 核心组成 | 特点与适用场景 |
| :--- | :--- | :--- |
| **普通调速阀** | 定差减压阀 + 节流阀（串联） | 负载变化时自动补偿，保持流量稳定。最基础型号，适用大多数需稳定速度的场合。 |
| **旁通型调速阀（溢流节流阀）** | 定差溢流阀 + 节流阀（并联） | 流量稳定性稍逊于普通型，但与定量泵配合时功率损耗小、发热少。适用对速度稳定性要求不高、更重节能的场景。 |
| **比例调速阀** | 比例电磁铁 + 调速阀 | 用电信号（电流/电压）控制阀开度，按比例无级调节流量。适用需远程控制或程序化自动控制的场合。 |
| **电磁调速阀** | 电磁换向阀 + 可调流量控制阀 | 通过电气信号实现速度“分段控制”（如慢速-快速两段式或三段式切换）。常见于塑胶射出成型机等设备。 |
| **行程调速阀** | 调速阀 + 行程限位器 | 执行元件运动至特定位置时，机械挡块撞击限位器，改变节流阀开度，实现速度自动切换。 |

## 二、按气动系统应用分类（气动常用）
气动系统中，调速阀多指**单向节流阀（速度控制阀）**，根据气流方向控制方式分两类：
| 类型 | 工作原理 | 特点 |
| :--- | :--- | :--- |
| **排气节流型** | 调节从气缸排气口排出的气体流量 | **最常用**回路方式。通过控制排气背压获得更平稳的气缸速度，稳定性好。 |
| **进气节流型** | 调节进入气缸的进气流量 | 速度控制难度较大、稳定性较差，主要用于单作用气缸、夹紧缸等对速度要求不高的场合。 |


## 三、其他辅助分类方式
- **按气体流动方向**：直通式（进出口同轴）、弯头式（进出口成90°）、万向式（出口方向可绕进口旋转）。
- **按阀体材质**：金属型（耐高温，适用焊接等恶劣环境）、树脂型（标准通用）。
- **按接口形式**：管接头型、法兰型、板式安装型等。
