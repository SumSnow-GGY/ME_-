

# Mask_Diff_bitNot 掩码取反及最大连通区域提取

## 算法简介

该算法用于对二值掩码进行预处理，主要包括：

1. 二值图像取反
2. 提取最大连通区域
3. 去除左右两侧突出的干扰区域
4. 得到稳定的检测 Mask

该算法主要用于颜色差分（Mask_Diff）之后的 ROI 提取，为后续边缘检测提供更加稳定的检测区域。

相比直接使用 Mask，本算法能够有效去除孤立噪点、小面积区域以及左右边缘毛刺，提高后续检测稳定性。

---

## 算法流程

```text
输入Mask
    │
    ▼
Bitwise Not
    │
    ▼
寻找所有轮廓
    │
    ▼
保留最大连通区域
    │
    ▼
计算水平投影
    │
    ▼
寻找左右有效边界
    │
    ▼
裁剪两侧突出区域
    │
    ▼
输出最终Mask
```

---

## 参数说明

| 参数 | 默认值 | 说明 | 调整建议 |
|------|--------|------|----------|
| MaskDiffNot | bool | 是否启用最大连通区域优化 | 根据检测模式开启 |
| RETR_EXTERNAL | - | 仅提取最外层轮廓 | 一般保持默认 |
| CHAIN_APPROX_SIMPLE | - | 轮廓压缩存储 | 一般保持默认 |
| Projection Threshold | 图像高度 × 20% | 判断有效列的投影阈值 | 可根据目标高度调整 |

# 核心步骤

## Step1：复制并取反 Mask

```cpp
cv::Mat mat_Mask_Diff_not = mat_Mask_Diff.clone();
cv::bitwise_not(mat_Mask_Diff_not, mat_Mask_Diff_not);
```

| 项目 | 内容 |
|------|------|
| 功能 | 对 Mask 进行取反 |
| 原理 | 黑白像素互换（0 ↔ 255） |
| 使用 clone() | 深拷贝，避免修改原始数据 |
| 工程说明 | 后续所有处理均基于复制图像，保证原始 Mask 数据安全 |

---

## Step2：提取所有外轮廓

```cpp
findContours(...)
```

| 项目 | 内容 |
|------|------|
| 功能 | 提取所有独立连通区域 |
| 模式 | `RETR_EXTERNAL` |
| 轮廓类型 | `CHAIN_APPROX_SIMPLE` |
| 工程说明 | 仅保留最外层轮廓，减少计算量 |

---

## Step3：寻找最大连通区域

遍历所有轮廓：

```cpp
contourArea(contours[i]);
```

记录面积最大的轮廓：

```cpp
maxContourIdx
```

随后重新生成 Mask：

```cpp
drawContours(...)
```

| 项目 | 内容 |
|------|------|
| 功能 | 保留面积最大的目标区域 |
| 判断依据 | `contourArea()` |
| 输出 | 单一连通区域 Mask |
| 工程价值 | 去除噪点、小面积误检区域，提高稳定性 |

---

## Step4：计算水平投影

统计每一列白色像素数量：

```cpp
colProjection[x]++
```

形成投影曲线：

```text
白色像素数量
↑
│          ██████
│      ███████████
│   ███████████████
└────────────────────► X
```

| 项目 | 内容 |
|------|------|
| 功能 | 统计每列有效像素数量 |
| 方法 | 每列累加白色像素 |
| 输出 | 一维投影数组 |
| 工程价值 | 用于分析目标宽度及边界位置 |

---

## Step5：寻找左右有效边界

定义有效列阈值：

```cpp
minProjection = rows × 20%
```

分别进行两次扫描：

左侧：

```text
左 → 右
```

找到第一列满足：

```text
Projection ≥ Threshold
```

右侧：

```text
右 → 左
```

找到最后一列满足：

```text
Projection ≥ Threshold
```

| 项目 | 内容 |
|------|------|
| 功能 | 去除左右两侧突出区域 |
| 判断依据 | 投影高度达到图像高度20% |
| 左边界 | 第一列有效投影 |
| 右边界 | 最后一列有效投影 |
| 工程价值 | 消除边缘毛刺及细长突出区域 |

---

## Step6：裁剪有效区域

根据左右边界：

```cpp
leftBound
rightBound
```

重新生成新的 Mask：

```cpp
mat_Clipped
```

仅保留：

```text
leftBound
      │
██████████████
██████████████
██████████████
          │
      rightBound
```

| 项目 | 内容 |
|------|------|
| 功能 | 去除左右无效区域 |
| 方法 | 按列复制有效区域 |
| 输出 | 裁剪后的 Mask |
| 工程价值 | 提高后续边缘检测准确率 |

---

## Step7：更新最终 Mask

```cpp
mat_Mask_Diff = mat_Mask_Diff_not.clone();
```

| 项目 | 内容 |
|------|------|
| 功能 | 保存最终处理结果 |
| 方法 | 深拷贝 |
| 工程说明 | 保证外部调用获得最终优化后的 Mask |

---

## 工程优化总结

| 优化项 | 实现方法 | 优点 | 工程价值 |
|--------|----------|------|----------|
| 深拷贝 | `clone()` | 不修改原始数据 | 提高程序安全性 |
| 图像取反 | `bitwise_not()` | 黑白区域互换 | 满足不同检测模式需求 |
| 最大连通区域 | `contourArea()` | 去除小面积噪点 | 提高 Mask 稳定性 |
| 外轮廓提取 | `RETR_EXTERNAL` | 减少计算量 | 忽略内部空洞干扰 |
| 水平投影 | 每列统计白色像素 | 获取目标宽度分布 | 后续边界裁剪依据 |
| 动态边界 | 图像高度20%阈值 | 自适应目标高度 | 避免固定宽度裁剪 |
| 区域裁剪 | 保留有效投影范围 | 去除左右毛刺 | 提高边缘检测稳定性 |

## 算法复杂度

设图像尺寸为 **H × W**。

| 操作 | 时间复杂度 | 说明 |
|------|------------|------|
| 图像取反 | O(H×W) | 遍历一次图像 |
| 轮廓提取 | O(H×W) | OpenCV 连通区域搜索 |
| 最大面积计算 | O(N) | N 为轮廓数量 |
| 水平投影 | O(H×W) | 每列统计白色像素 |
| 左右边界搜索 | O(W) | 两次线性扫描 |
| 区域裁剪 | O(H×W) | 拷贝有效区域 |

整体复杂度约为：

```text
O(H × W)
```

属于线性复杂度，能够满足工业视觉实时处理需求。

## 工程经验

- `clone()` 比直接赋值更安全，可避免 OpenCV 引用计数导致原始数据被意外修改。
- 使用 **最大连通区域** 而非面积阈值过滤，对不同尺寸目标具有更好的自适应性。
- `RETR_EXTERNAL` 仅提取最外层轮廓，可避免目标内部孔洞影响区域提取。
- 水平投影裁剪比固定宽度裁剪更稳定，可适应不同宽度的产品。
- 投影阈值设置为 **图像高度的20%**，能够过滤细小毛刺，但对于高度较小或存在缺口的目标，可适当降低至 **10%~15%**。
- 当前裁剪仅处理左右方向的突出区域，若目标上下也存在明显毛刺，可进一步结合垂直投影实现四周自动裁剪。

## 适用场景

- 颜色差分后的 ROI 提取
- 连通区域筛选
- 二值图像预处理
- 工业视觉 Mask 优化
- 条纹、标签、薄膜等目标区域提取
- 后续边缘检测、模板匹配、缺陷检测前处理


```cpp
void Inspect_Color::Mask_Diff_bitNot()
{
    if (!mat_Mask_Diff.empty())
    {
        // 使用 clone() 进行深拷贝，避免修改原始数据
        cv::Mat mat_Mask_Diff_not = mat_Mask_Diff.clone();
        cv::bitwise_not(mat_Mask_Diff_not, mat_Mask_Diff_not);
        // 保留mat_Mask_Diff_not最大面积的联通区域

        if(m_Parameters_Color.m_iFlag_MaskDiffNot)
        {
            // ========== 保留最大面积的连通区域 ==========
            std::vector<std::vector<cv::Point>> contours;
            std::vector<cv::Vec4i> hierarchy;
            // 查找所有轮廓
            findContours(mat_Mask_Diff_not, contours, hierarchy,
                        cv::RETR_EXTERNAL, cv::CHAIN_APPROX_SIMPLE);
            if (!contours.empty())
            {
                // 找到面积最大的轮廓
                int maxContourIdx = -1;
                double maxArea = 0;

                for (size_t i = 0; i < contours.size(); i++)
                {
                    double area = contourArea(contours[i]);
                    if (area > maxArea)
                    {
                        maxArea = area;
                        maxContourIdx = static_cast<int>(i);
                    }
                }
                // 创建只包含最大轮廓的掩码
                cv::Mat mat_Largest_Region = cv::Mat::zeros(mat_Mask_Diff_not.size(), CV_8UC1);
                if (maxContourIdx >= 0)
                {
                    drawContours(mat_Largest_Region, contours, maxContourIdx,
                                cv::Scalar(255), cv::FILLED);
                    if (maxContourIdx >= 0)
                    {
                        drawContours(mat_Largest_Region, contours, maxContourIdx,
                                    cv::Scalar(255), cv::FILLED);

                        // ========== 剔除左右两侧突出区域 ==========
                        // 1. 计算水平投影（每列的白色像素数）
                        std::vector<int> colProjection(mat_Largest_Region.cols, 0);
                        for (int x = 0; x < mat_Largest_Region.cols; x++)
                        {
                            for (int y = 0; y < mat_Largest_Region.rows; y++)
                            {
                                if (mat_Largest_Region.at<uchar>(y, x) == 255)
                                {
                                    colProjection[x]++;
                                }
                            }
                        }

                        // 2. 找到有效列的范围
                        int leftBound = 0, rightBound = mat_Largest_Region.cols - 1;
                        int minProjection = mat_Largest_Region.rows * 0.2;  // 阈值：至少20%高度有像素
                        
                        // 从左向右找第一个有效列
                        while (leftBound < mat_Largest_Region.cols && colProjection[leftBound] < minProjection)
                        {
                            leftBound++;
                        }
                        
                        // 从右向左找最后一个有效列
                        while (rightBound >= 0 && colProjection[rightBound] < minProjection)
                        {
                            rightBound--;
                        }

                        // 3. 创建裁剪后的区域
                        cv::Mat mat_Clipped = cv::Mat::zeros(mat_Largest_Region.size(), CV_8UC1);
                        if (leftBound <= rightBound)
                        {
                            // 将中间有效区域复制到新图像
                            for (int y = 0; y < mat_Largest_Region.rows; y++)
                            {
                                for (int x = leftBound; x <= rightBound; x++)
                                {
                                    mat_Clipped.at<uchar>(y, x) = mat_Largest_Region.at<uchar>(y, x);
                                }
                            }
                        }
                        
                        mat_Largest_Region = mat_Clipped;
                    }
                }
                // 将最大区域复制回原图像
                mat_Mask_Diff_not = mat_Largest_Region;
            }
            else
            {
                // 如果没有找到任何轮廓，清空图像
                mat_Mask_Diff_not.setTo(cv::Scalar(0));
            }
        }
        mat_Mask_Diff = mat_Mask_Diff_not.clone();
    }
}
```
