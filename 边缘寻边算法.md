

# 双边滤波 + 投影梯度单边缘检测算法

## 算法简介

该算法用于检测 ROI 区域左右边缘的位置，适用于具有明显垂直边缘特征的工业视觉检测场景。

算法首先利用双边滤波去除噪声并保留边缘，
然后计算 X 方向 Sobel 梯度，通过列投影将二维梯度压缩为一维信号，
再利用高斯平滑和一阶导数增强边缘特征，最后根据不同模式寻找左右边缘位置。

相比直接在二维图像上寻找边缘，该方法具有以下优点：

- 降低二维图像噪声影响
- 利用投影提高边缘稳定性
- 对长条状目标具有较高鲁棒性
- 计算量较小，适合实时检测
---

## 算法流程

```text
输入ROI
    │
    ▼
双边滤波(Bilateral Filter)
    │
    ▼
Sobel(X方向梯度)
    │
    ▼
列投影(Reduce Sum)
    │
    ▼
平均梯度
    │
    ▼
高斯平滑
    │
    ▼
一阶导数
    │
    ├──────────────┐
    ▼              ▼
正常模式      MaskDiffNot模式
    │              │
寻找左右最大梯度   寻找所有局部峰值
    │              │
    └──────┬───────┘
           ▼
      输出左右边缘
```

---

## 参数说明

| 参数 | 默认值 | 说明 | 调整建议 |
|------|--------|------|----------|
| Bilateral Diameter | 10 | 双边滤波窗口大小 | 噪声较大可适当增大 |
| SigmaColor | 20 | 颜色空间权重 | 越大平滑越明显 |
| SigmaSpace | 20 | 空间距离权重 | 根据目标尺寸调整 |
| Gaussian Sigma | 7.0 | 投影平滑参数 | 噪声较大时适当增大 |
| Sobel Kernel | 3 | Sobel卷积核大小 | 一般保持3即可 |

# 核心步骤

## Step1：双边滤波预处理

```cpp
bilateralFilter(mat_In, b_ROI, 10, 20, 20);
```

| 项目 | 内容 |
|------|------|
| 功能 | 去除噪声，同时保留边缘信息 |
| 原理 | 综合考虑空间距离和灰度差异进行滤波 |
| 优点 | 相比高斯滤波不会模糊边缘 |
| 工程说明 | 工业视觉中边缘定位通常优先采用双边滤波 |

---

## Step2：计算X方向梯度

```cpp
Sobel(b_ROI, grad, CV_16S, 1, 0);
```

| 项目 | 内容 |
|------|------|
| 功能 | 提取图像左右边缘 |
| X方向 | dx=1，dy=0 |
| 输出类型 | CV_16S，避免梯度溢出 |
| 工程说明 | 左右边缘检测通常只需要X方向梯度 |

---

## Step3：列投影

```cpp
reduce(grad, proj, 0, REDUCE_SUM, CV_32F);
proj /= RectHeight;
```

| 项目 | 内容 |
|------|------|
| 功能 | 将二维梯度压缩成一维梯度曲线 |
| 方法 | 每列梯度求和 |
| 除以高度 | 求取平均梯度，避免ROI高度影响结果 |
| 工程价值 | 大幅降低噪声，提高检测稳定性 |

二维梯度：

```
↑
│ █ █ █ █
│ █ █ █ █
│ █ █ █ █
└────────────►
```

列投影后：

```
▲
│
│      ▲
│     ▲ ▲
│    ▲   ▲
└────────────► X
```

---

## Step4：投影平滑

```cpp
GaussianBlur(proj, projSmooth, Size(0,0), 7.0);
```

| 项目 | 内容 |
|------|------|
| 功能 | 去除投影曲线毛刺 |
| 原理 | 高斯低通滤波 |
| 参数 | σ=7 |
| 工程经验 | σ越大，曲线越平滑，但定位精度略有下降 |

---

## Step5：计算投影一阶导数

```cpp
Sobel(projSmooth, grad_Smooth, CV_32F, 1, 0, 3);
```

| 项目 | 内容 |
|------|------|
| 功能 | 增强投影曲线变化位置 |
| 输出 | 一维梯度曲线 |
| 作用 | 边缘位置对应梯度峰值 |
| 工程价值 | 比直接分析投影更加稳定 |

---

## Step6：MaskDiffNot 模式

该模式主要用于颜色取反后的边缘检测。

首先计算梯度绝对值：

```cpp
gradAbs = abs(grad_Smooth);
```

然后计算动态阈值：

```cpp
threshold = maxGradient × 0.2
```

随后寻找局部极大值：

```cpp
v > 左侧3个点
v > 右侧3个点
```

满足条件即认为是一个有效边缘峰值。

最终：

- 第一个峰值作为左边缘
- 最后一个峰值作为右边缘

### 特点

| 项目 | 内容 |
|------|------|
| 阈值 | 最大梯度20% |
| 峰值条件 | 连续7点局部最大 |
| 左边缘 | 第一个峰值 |
| 右边缘 | 最后一个峰值 |
| 优点 | 适用于存在多个连续边缘的情况 |

---

## Step7：普通模式

普通模式假设目标位于图像中心附近。

首先计算中心位置：

```cpp
center = cols / 2
```

然后：

左侧：

- 从中心向左扫描
- 寻找最大正梯度

右侧：

- 从中心向右扫描
- 寻找最大负梯度（取绝对值比较）

### 特点

| 项目 | 内容 |
|------|------|
| 搜索方式 | 中心向两侧搜索 |
| 左边缘 | 最大正梯度 |
| 右边缘 | 最大负梯度 |
| 优点 | 速度快、误检少 |
| 缺点 | 要求目标位于图像中央附近 |

---

## Step8：保存检测结果

检测完成后保存左右边缘坐标：

```cpp
maxpoints1
maxpoints2
```

同时记录梯度值：

```cpp
m_Current_Grandient[]
```

供后续质量判断或缺陷分析使用。

## 工程优化总结

| 优化项 | 实现方法 | 优点 | 工程价值 |
|--------|----------|------|----------|
| 双边滤波 | Bilateral Filter | 保留边缘同时去噪 | 提高边缘定位精度 |
| 列投影 | Reduce Sum | 二维转一维 | 降低噪声，提高稳定性 |
| 平均梯度 | 除以ROI高度 | 消除高度影响 | 不同ROI尺寸结果一致 |
| 高斯平滑 | GaussianBlur | 去除投影毛刺 | 提高峰值稳定性 |
| 一阶导数 | Sobel | 增强边缘变化 | 峰值更加明显 |
| 动态阈值 | 最大值20% | 自动适应不同图像 | 无需固定阈值 |
| 局部峰值判断 | 连续7点比较 | 去除毛刺峰值 | 降低误检率 |
| 双模式检测 | MaskDiffNot/Normal | 适应不同图像类型 | 提高算法通用性 |

## 算法复杂度

设图像大小为 **H × W**。

| 操作 | 时间复杂度 | 说明 |
|------|------------|------|
| 双边滤波 | O(H×W×k²) | 计算量最大 |
| Sobel梯度 | O(H×W) | X方向梯度 |
| 列投影 | O(H×W) | 每列求和 |
| 高斯平滑 | O(W) | 一维曲线处理 |
| 一阶导数 | O(W) | 一维Sobel |
| 峰值搜索 | O(W) | 遍历梯度曲线 |

整体复杂度约为：

```text
O(H×W×k²)
```

主要耗时集中在双边滤波阶段。

## 工程经验

- ROI 应尽量裁剪到目标附近，可减少双边滤波计算量。
- 双边滤波虽然效果好，但速度较慢；实时性要求较高时可考虑导向滤波（Guided Filter）或高斯滤波作为替代。
- 列投影前使用 Sobel 提取梯度，可显著增强边缘响应，避免直接投影受灰度变化影响。
- 投影曲线进行高斯平滑后再求导，比直接对原始投影求导更稳定，可有效抑制噪声引起的伪峰值。
- MaskDiffNot 模式采用 **20% 最大梯度** 作为动态阈值，对不同亮度图像具有较好的自适应能力；若噪声较多，可适当提高阈值比例（如 30%）。
- 局部峰值采用 **±3 个采样点** 比较，可有效滤除单点毛刺；若目标边缘较宽，可适当增加比较窗口。
