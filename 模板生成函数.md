# Generic ShapeModel 模板建模（genGeoMatchModel）

## dll文件导入


## 算法简介

`genGeoMatchModel()` 用于生成 Halcon Generic ShapeModel 模板模型。

整个流程包括 **输入检查、Mask 区域处理、图像预处理、ShapeModel 参数配置、模板训练以及模板轮廓绘制** 六个步骤。

为了提高模板匹配的稳定性，算法支持：

- ROI 建模
- 涂抹区域（Mask）屏蔽
- 图像形态学平滑
- 尺度匹配
- 角度匹配
- 自动金字塔层数
- 自动优化模板

最终生成可用于 Shape Matching 的模板模型，并将模板轮廓绘制到显示图像。

---

## 算法流程

```text
输入ROI
    │
    ▼
参数检查
    │
    ▼
Mask区域处理
    │
    ▼
图像预处理
    │
    ▼
生成Halcon图像
    │
    ▼
ShapeModel参数配置
    │
    ▼
模板训练
    │
    ▼
获取模板轮廓
    │
    ▼
OpenCV绘制模板
```

## 输入参数

| 参数 | 类型 | 说明 |
|------|------|------|
| mat_In | cv::Mat | 输入 ROI 图像 |
| roi | cv::Rect | 模板所在区域 |

---

## 成员变量

| 变量 | 说明 |
|------|------|
| ModelID | ShapeModel 模板句柄 |
| mat_Mask_daub | 涂抹 Mask |
| mat_Temp_Edge_Dst | 预处理结果 |
| m_Mat_Dst_Color | 模板显示图像 |
| m_min_contrast | ShapeModel 最低对比度 |
| m_Smoothing_Coe | 图像平滑系数 |
| m_MatchScale | 模板缩放比例 |
| m_Angle_uplimit | 最大旋转角度 |

---

## 前置条件

| 检查项 | 原因 |
|---------|------|
| 输入图像不能为空 | 防止 OpenCV 处理异常 |
| m_min_contrast > 0 | Halcon 对比度参数非法会导致建模失败 |
| m_Smoothing_Coe > 0 | 灰度形态学窗口不能为 0 |
| ROI 必须合法 | 防止 ROI 越界导致程序崩溃 |

# 核心步骤

## Step1：参数检查

首先判断输入是否合法。

```cpp
if(mat_In.empty())
    return;

if(m_min_contrast == 0)
    return;

if(m_Smoothing_Coe == 0)
    return;
```

| 项目 | 内容 |
|------|------|
| 功能 | 判断建模条件是否满足 |
| 优点 | 避免非法参数导致程序异常 |
| 工程说明 | 建议所有建模函数均保留参数检查 |

---

## Step2：Mask 区域处理

如果存在涂抹 Mask，则首先裁剪 ROI 内的 Mask，并转换为 Halcon Region。

随后完成：

- Mask 区域生成
- ROI 区域生成
- Mask 与 ROI 求交集
- ROI 去除 Mask

最终得到真正参与建模的区域。

流程如下：

```text
Mask
    │
    ▼
GenRegionPoints
    │
    ▼
ROI Rectangle
    │
    ▼
Intersection
    │
    ▼
Difference
    │
    ▼
ModelRegionFiltered
```

| 项目 | 内容 |
|------|------|
| 功能 | 去除用户涂抹区域 |
| 优点 | 被屏蔽区域不会参与建模 |
| 使用函数 | GenRegionPoints、Intersection、Difference |
| 工程价值 | 避免干扰区域进入模板，提高匹配稳定性 |

---

## Step3：图像预处理

首先执行边缘增强：

```cpp
Detect_SmoothEdge(...)
```

随后转换为 Halcon 图像：

```cpp
Shape_ConvertImage(...)
```

最后执行灰度开闭运算：

```cpp
GrayOpeningShape(...)
GrayClosingShape(...)
```

| 项目 | 内容 |
|------|------|
| 功能 | 去除噪声，提高边缘质量 |
| 开运算 | 去除小面积亮噪声 |
| 闭运算 | 填补边缘小孔洞 |
| 结构元素 | Octagon（八边形） |
| 工程价值 | 提高模板轮廓连续性，减少误匹配 |

---

## Step4：ShapeModel 参数配置

创建 Generic ShapeModel：

```cpp
CreateGenericShapeModel(...)
```

随后设置模板参数。

### ShapeModel 参数说明

| 参数 | 作用 | 当前设置 | 工程说明 |
|------|------|----------|----------|
| scale_row_min | 最小行缩放比例 | 根据 m_MatchScale 自动计算 | 支持尺度变化 |
| scale_row_max | 最大行缩放比例 | 根据 m_MatchScale 自动计算 | 支持尺度变化 |
| scale_column_min | 最小列缩放比例 | 自动计算 | 保持 XY 一致缩放 |
| scale_column_max | 最大列缩放比例 | 自动计算 | 保持 XY 一致缩放 |
| num_levels | 金字塔层数 | 0（自动） | Halcon 自动选择最佳层数 |
| angle_start | 起始旋转角 | -m_Angle_uplimit | 支持旋转匹配 |
| angle_end | 结束旋转角 | +m_Angle_uplimit | 支持旋转匹配 |
| angle_step | 角度步长 | auto | 自动优化 |
| optimization | 模型优化方式 | auto | 自动优化模板 |
| metric | 极性模式 | ignore_local_polarity | 忽略局部极性变化 |
| contrast_low | 最低对比度 | m_min_contrast | 提取弱边缘 |
| contrast_high | 高对比度 | m_min_contrast+4 | 保证稳定建模 |

---

## Step5：模板训练

完成参数配置后开始训练模板。

```cpp

```

训练完成后获得：

```text

```

该对象用于后续：

- FindGenericShapeModel()
- 模板保存

| 项目 | 内容 |
|------|------|
| 功能 | 建立 ShapeModel |
| 输出 | ModelID |
| 工程说明 | 一个模板仅需训练一次，可重复使用 |

## Step6：模板轮廓绘制

训练完成后获取模板轮廓：

```cpp
GetShapeModelContours(...)
```

随后：

- 遍历所有轮廓
- 获取轮廓坐标
- 转换为 OpenCV Point
- 绘制到显示图像

流程如下：

```text

    │
    ▼
GetShapeModelContours
    │
    ▼
GetContourXld
    │
    ▼
Point转换
    │
    ▼
OpenCV Polyline
```

| 项目 | 内容 |
|------|------|
| 功能 | 显示模板边缘 |
| 输出 | OpenCV Polyline |
| 工程价值 | 方便调试模板质量及建模范围 |

## 工程优化总结

| 优化项 | 实现方法 | 工程价值 |
|--------|----------|----------|
| 参数检查 | 输入合法性判断 | 防止非法参数建模失败 |
| ROI 建模 | ReduceDomain | 减少无关区域干扰 |
| Mask 去除 | Difference | 屏蔽用户指定区域 |
| 图像平滑 | Gray Opening + Closing | 提高轮廓连续性 |
| 自动尺度 | scale_row/column | 支持目标缩放 |
| 自动角度 | angle_start/end | 支持旋转匹配 |
| 自动金字塔 | num_levels=0 | 提高匹配效率 |
| 自动优化 | optimization=auto | 减少模板冗余 |
| 极性忽略 | ignore_local_polarity | 适应亮暗变化 |
| 双对比度 | contrast_low/high | 提高模板稳定性 |
| 异常保护 | try-catch | 提高程序健壮性 |

## 算法复杂度

设 ROI 图像大小为 **H × W**。

| 操作 | 时间复杂度 | 说明 |
|------|------------|------|
| Mask 转换 | O(H×W) | 遍历 ROI |
| 图像预处理 | O(H×W) | 开闭运算 |
| ReduceDomain | O(H×W) | ROI 裁剪 |
| ShapeModel 建模 | 与模板复杂度相关 | Halcon 内部实现 |
| 模板轮廓绘制 | O(N) | N 为轮廓点数 |

整体耗时主要集中在 **TrainGenericShapeModel()**，其耗时与模板尺寸、边缘数量、金字塔层数及旋转范围密切相关。

---

## 工程经验

- 建议 ROI 尽量裁剪至目标附近，可显著缩短模板训练时间。
- `ignore_local_polarity` 能提高亮暗变化环境下的匹配鲁棒性，但对于依赖极性的目标，可根据需求改为严格匹配模式。
- `contrast_low` 不宜设置过低，否则容易将噪声边缘纳入模板；过高则可能丢失弱边缘，应结合目标对比度调整。
- `GrayOpeningShape()` 与 `GrayClosingShape()` 可有效平滑边缘，但平滑系数过大可能导致细小特征丢失。
- 模板训练完成后建议保存 `ModelID`，避免重复建模，提高软件启动和运行效率。

## 源代码
```cpp
void widget_shaoematching_editing::genGeoMatchModel(cv::Mat& mat_In, cv::Rect& roi)
{

    try{
        if(mat_In.empty())   // 这里待修改——输入的图像分类：截取的图像区域而不是整个图像
            return;
        if(m_min_contrast == 0)
            return;
        if(m_Smoothing_Coe == 0)
            return;

        // 涂抹区域裁剪
        cv::Mat roiMask;
        HObject MaskRegion, ModelRegion, MaskInROI, ModelRegionFiltered;
        std::vector<Hlong> rows;
        std::vector<Hlong> cols;

        if(!mat_Mask_daub.empty())
        {
            roiMask = mat_Mask_daub(roi);
            // 裁剪Halcon图像
            for (int y = 0; y < roiMask.rows; y++)
            {
                const uchar* p = roiMask.ptr<uchar>(y);
                for (int x = 0; x < roiMask.cols; x++)
                {
                    if (p[x] != 0)
                    {
                        rows.push_back(y);
                        cols.push_back(x);
                    }
                }
            }

            if (!rows.empty())
            {
                GenRegionPoints(&MaskRegion, HTuple(rows.data(), rows.size()),HTuple(cols.data(), cols.size()));
            }
            else
            {
                GenEmptyObj(&MaskRegion);
            }

            GenRectangle2(&ModelRegion, roi.height/2, roi.width/2, 0, roi.width/2, roi.height/2);
            Intersection(MaskRegion, ModelRegion, &MaskInROI);
            Difference(ModelRegion, MaskInROI, &ModelRegionFiltered);
        }

        // 图像预处理
        Detect_SmoothEdge(mat_In , mat_Temp_Edge_Dst);

        cv::Mat mat_Temp_Combine_Dst;
        mat_Temp_Combine_Dst = mat_Temp_Edge_Dst.clone();

        cv::Mat drawImg = mat_In.clone();   // roiMat（局部图）

        HObject Image;
        Shape_ConvertImage(mat_In, &Image);

        HalconCpp::GrayOpeningShape(Image, &Image, m_Smoothing_Coe, m_Smoothing_Coe, "octagon");
        HalconCpp::GrayClosingShape(Image, &Image, m_Smoothing_Coe, m_Smoothing_Coe, "octagon");

        HObject TemplateImage;
        HalconCpp::ReduceDomain(Image, ModelRegionFiltered, &TemplateImage);
        HTuple Contrast;
        Contrast[0] = m_min_contrast;     // 数据类型必须使用 HTuple 类型
        // ***对比度，自动调整
        Contrast[1] = m_min_contrast + 4; // 数据类型必须使用 HTuple 类型
        // 创建模板
        HalconCpp::CreateGenericShapeModel(&ModelID);
        // 设置参数
        HalconCpp::SetGenericShapeModelParam(ModelID, "scale_row_min", HTuple(1));
        HalconCpp::SetGenericShapeModelParam(ModelID, "scale_row_max", HTuple(1));
        if(m_MatchScale < 1.0)
        {
            HalconCpp::SetGenericShapeModelParam(ModelID, "scale_row_min", HTuple(m_MatchScale));
            HalconCpp::SetGenericShapeModelParam(ModelID, "scale_row_max", HTuple(1 + abs(m_MatchScale - 1.0)));

            HalconCpp::SetGenericShapeModelParam(ModelID, "scale_column_min", HTuple(m_MatchScale));
            HalconCpp::SetGenericShapeModelParam(ModelID, "scale_column_max", HTuple(1 + abs(m_MatchScale - 1.0)));
        }
        else
        {
            HalconCpp::SetGenericShapeModelParam(ModelID, "scale_row_min", HTuple(1 - abs(m_MatchScale - 1.0)));
            HalconCpp::SetGenericShapeModelParam(ModelID, "scale_row_max", HTuple(m_MatchScale));

            HalconCpp::SetGenericShapeModelParam(ModelID, "scale_column_min", HTuple(1 - abs(m_MatchScale - 1.0)));
            HalconCpp::SetGenericShapeModelParam(ModelID, "scale_column_max", HTuple(m_MatchScale));
        }

        HalconCpp::SetGenericShapeModelParam(ModelID, "num_levels", HTuple(0));
        HalconCpp::SetGenericShapeModelParam(ModelID, "angle_start", HTuple(-m_Angle_uplimit).TupleRad());
        HalconCpp::SetGenericShapeModelParam(ModelID, "angle_end", HTuple(m_Angle_uplimit).TupleRad());
        HalconCpp::SetGenericShapeModelParam(ModelID, "angle_step", "auto");
        HalconCpp::SetGenericShapeModelParam(ModelID, "optimization", "auto");
        HalconCpp::SetGenericShapeModelParam(ModelID, "metric", "ignore_local_polarity");
        HalconCpp::SetGenericShapeModelParam(ModelID, "contrast_low", Contrast[0]); // min_contrast 自动计算阈值报错HalconError
        HalconCpp::SetGenericShapeModelParam(ModelID, "contrast_high",Contrast[1]);
        HalconCpp::TrainGenericShapeModel(TemplateImage, ModelID);

        HObject ModelContours;
        GetShapeModelContours(&ModelContours, ModelID, 1);

        HTuple numCont;
        CountObj(ModelContours, &numCont);
        // 绘制轮廓图像
        if (numCont.I() > 0 && !mat_In.empty())
        {
            for (int k = 1; k <= numCont.I(); ++k)
            {
                HObject oneCont;
                SelectObj(ModelContours, &oneCont, k);

                HTuple rows, cols;
                GetContourXld(oneCont, &rows, &cols);

                if (rows.Length() == 0 || cols.Length() == 0) continue;

                std::vector<cv::Point> pts;
                pts.reserve(rows.Length());

                for (int i = 0; i < rows.Length(); ++i)
                {
                    int x = (int)cols[i].D() + roi.x + roi.width  / 2.0;
                    int y = (int)rows[i].D() + roi.y + roi.height / 2.0;
                    pts.emplace_back(x, y);
                }
                polylines(m_Mat_Dst_Color, pts, false, cv::Scalar(255,255,0), 1, cv::LINE_AA);
            }
        }
    }
    catch(const cv::Exception& e){
        qDebug()<< "[OpenCV Error] " << e.what();
    }
    catch(const HalconCpp::HException& e)
    {
        qDebug()<< "[Halcon Error] " << e.ErrorMessage().Text();
    }
     catch (const std::exception& e)
    {
        qDebug()<< "[STD Error] " << e.what();
    }
    catch (...)
    {
        qDebug()<< "[Unknown Error] genGeoMatchModel crashed";
    }
}
```
