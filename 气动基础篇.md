### 0. 压力概念：
	**空气是可以被压缩的。压缩后，即可产生气压力了**

压力单位换算
![输入图片说明](/imgs/2026-06-16/R3CKKHhsXcUAxbik.png)
- 补充：

| 单位名称 | 符号 | 与 Pa 的换算关系 | 与标准大气压的关系 | 主要应用领域 |
| :--- | :---: | :--- | :--- | :--- |
| 帕斯卡 | Pa | 1 Pa = 1 N/m² | 1 atm = 101325 Pa | 国际单位制（SI）基本压力单位 |
| 千帕 | kPa | 1 kPa = 1000 Pa | 1 atm = 101.325 kPa | 气象、工程常用 |
| 兆帕 | MPa | 1 MPa = 10⁶ Pa | 1 atm ≈ 0.1013 MPa | 空压机、液压、材料力学 |
| 巴 | bar | 1 bar = 10⁵ Pa | 1 atm = 1.01325 bar | 欧洲常用、空压机基准状态压力 |
| 标准大气压 | atm | 1 atm = 101325 Pa | 1 atm | 化学、物理经典定义 |
| 托 | Torr | 1 Torr ≈ 133.322 Pa | 1 atm = 760 Torr（精确） | 真空技术、半导体、医学 |
| 毫米汞柱 | mmHg | 1 mmHg ≈ 133.322 Pa | 1 atm = 760 mmHg | 血压测量、气象（传统） |
| 磅力/平方英寸 | psi | 1 psi ≈ 6894.76 Pa | 1 atm ≈ 14.696 psi | 美国、英制国家常用 |

压力划分
![输入图片说明](/imgs/2026-06-16/TRzIS2gN8AZzGIcI.png)

**绝对压力=相对压力（表压力）+大气压力**
	
	波意耳定律

$P_1 V_1 = P_2 V_2$ 
	
	波意耳定律(带温度)
$\frac{P_1 V_1}{T_1} = \frac{P_2 V_2}{T_2}$

### 总结提醒
| 物理量 | 符号 | 说明 |
| :--- | :--- | :--- |
| 表压力 | $P_g$ | 压力表读数，日常口语中的“压力” |
| 大气压 | $P_{atm}$ | 通常取 $0.1 \, \text{MPa}$ |
| 绝对压力 | $P = P_g + P_{atm}$ | 气体定律中必须使用 |

### 空气状态：标准状态 vs 基准状态
| 对比项 | 标准状态 | 基准状态 |
| :--- | :--- | :--- |
| **主要用途** | 统一气体体积计量，便于比较 | 计算能量、效率、设备性能 |
| **温度** | 0 ℃ （273.15 K） | 20 ℃ （293.15 K） |
| **压力（绝压）** | 101.325 kPa （1 atm） | 100 kPa （1 bar） |
| **湿度** | 通常不考虑（干空气） | 明确给定（通常为 0%） |
| **典型应用场景** | 气体流量计、排放计算、化学计量 | 空压机性能曲线、比功率、内燃机测试 |
| **体积流量单位** | Nm³/h（标准立方米/小时） | m³/h 或 FAD（实际容积流量） |
| **一句话记忆** | **冷、准** → 比较“有多少” | **暖、整** → 评价“多能干” |
